# Keep Going — Focus Timer

A 75 Hard-inspired Pomodoro focus timer. Build your wall, one brick at a time.

**Live:** [keepgoing.club](https://keepgoing.club)

## Features

- 🧱 Brick-based progress — every session is a brick laid
- 🎯 Project battle plans with daily brick budgets
- 🧠 Brain.fm-style neural entrainment audio (Gamma 40Hz, Alpha 10Hz, Theta 6Hz, Beta Binaural 18Hz)
- 💾 All data persists locally in your browser
- 📱 Mobile-first design

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + DaisyUI
- Web Audio API (no external audio files)

## Dev

```bash
npm install
npm run dev
```

## Deploy

Deployed via Vercel. Every push to `main` auto-deploys.
